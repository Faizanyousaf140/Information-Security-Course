# Information Security Course

A comprehensive repository for Information Security course materials, designed for students, learners, and professionals who want to understand the fundamentals and advanced concepts of cybersecurity. This repository includes lecture slides, reference papers, notes, assignments, and more, covering all essential topics in Information Security.

# Overview

Information Security (InfoSec) is a critical field in computer science and IT, focusing on protecting information systems against threats, vulnerabilities, and attacks. This repository provides structured learning materials to build strong foundational knowledge and practical understanding.

### 🎯 Topics Covered

Introduction to Information Security

Importance of InfoSec

Security objectives: Confidentiality, Integrity, Availability

Threats, vulnerabilities, and risk assessment

Cryptography

Symmetric cryptography: AES, DES, block and stream ciphers

Asymmetric cryptography: RSA, ECC

Hash functions, digital signatures

Key management and security protocols

Network Security

Firewalls, IDS/IPS, VPNs

TCP/IP security considerations

Network attacks: sniffing, spoofing, MITM

Network security protocols (SSL/TLS, IPSec)

Authentication & Access Control

User authentication mechanisms

Password policies, multi-factor authentication

Access control models: DAC, MAC, RBAC

Application & Web Security

Common vulnerabilities: SQLi, XSS, CSRF

Secure coding practices

Web application security frameworks

Attack Vectors & Social Engineering

Phishing, spoofing, replay, masquerade

Buffer overflows, DoS/DDoS attacks

Security awareness & mitigation strategies

Security Standards & Frameworks

NIST Cybersecurity Framework (CSF)

ISO/IEC 27001

Risk management and governance
